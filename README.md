# рџ“– Project Overview

Investment Portfolio Risk Analytics is an end-to-end data analytics project developed to simulate how financial institutions manage investment portfolios and make data-driven decisions.

The project begins with generating realistic financial data using Python, followed by loading the data into SQL Server through an ETL process. Business-focused SQL views were created to prepare analytical datasets, which were then visualized using interactive Power BI dashboards.

The final solution enables executives and analysts to monitor portfolio performance, client analytics, investment distribution, and trading activity through a centralized reporting system.

Investment firms manage thousands of clients, portfolios, financial assets, and daily transactions. Without centralized reporting, it becomes difficult to monitor investment performance, understand client behavior, evaluate portfolio distribution, and analyze trading activities.
